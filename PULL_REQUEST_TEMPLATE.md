## Changes Made
- Internationalization (i18n) system implemented with runtime language detection
- Created locale files for English, Chinese, and French
- Updated ChatInput, Button, and SettingsPanel components to use translations
- Added parameterization and pluralization support
- Created comprehensive documentation for the i18n system

## Motivation and Context
This PR introduces a world-class text management system that allows for easy localization of the chrome-chat application. The system automatically detects the user's browser language and serves appropriate translations.
